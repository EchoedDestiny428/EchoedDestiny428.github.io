# EchoedDestiny428.github.io

A Jekyll-powered portfolio website.

## Quick Setup

**If you don't have Ruby installed yet**, see [INSTALLATION.md](INSTALLATION.md) for detailed Windows installation instructions.

Once Ruby is installed:

1. Install Bundler: `gem install bundler`
2. Install dependencies: `bundle install`
3. Run Jekyll: `bundle exec jekyll serve`
4. Open your browser to `http://localhost:4000`

## Building

To build the site: `bundle exec jekyll build`

The site will be generated in the `_site` directory.

## Deployment

This repository is set up for GitHub Pages. Simply push your changes to the main branch and GitHub Pages will automatically build and deploy your site.
